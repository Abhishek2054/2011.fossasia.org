fossasia11
==========
This is Jekyll-based website, for FOSSASIA'11 Conference Event, created during Google Code-In 2014 contest. 

Installation
---
Use the following commands - 

```
$ git clone http://github.com/fossasia/fossasia11
$ cd fossasia11
$ jekyll serve --baseurl=''

```
